# Kerykeion

Kerykeion is a python library for Astrology.


```python

>>> from kerykeion.astrocore import AstroData, Calculator
>>> from kerykeion.output import output

>>> kanye = Calculator("Kanye", 1977, 6, 8, 8, 45, "Atlanta")
>>> print(output(kanye))

```
