# Release Notes v5.5.0

## Brief Recap

-   Introduced **`next_return_from_date`** method to allow precise return calculations from a specific day, month, and year.
-   Added **`day` parameter validation** using `calendar.monthrange`.
-   Added **Aspect Icons on Chart Lines**: New `show_aspect_icons` option in `ChartDrawer` (default `True`) displays the aspect symbol at the center of aspect lines. Includes collision detection to prevent clutter.
