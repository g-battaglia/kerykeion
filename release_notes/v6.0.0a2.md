## BCE Date Support

Pass negative years to `from_birth_data()`:

```python
from kerykeion import AstrologicalSubjectFactory

subject = AstrologicalSubjectFactory.from_birth_data(
    name="Ancient Greece",
    year=-500, month=3, day=21, hour=12, minute=0,
    lat=37.9838, lng=23.7275, tz_str="Europe/Athens",
    online=False,
)
```

**Year convention:** astronomical year numbering (0 = 1 BCE, -1 = 2 BCE, -44 = 45 BCE, etc.), matching the native convention of the Swiss Ephemeris.

**Timezone handling:** for dates before 1 AD, input time is interpreted as Local Mean Time (LMT). The UTC offset is derived from longitude (1 hour per 15°). Standardized time zones did not exist before the 19th century, so LMT is the historically correct approach.

**Calendar:** all BCE dates use the Julian calendar (`SE_JUL_CAL`), which predates the Gregorian reform of 1582.
