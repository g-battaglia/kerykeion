# Release Notes v5.11.0

## Highlights

-   **Modern Chart Style**: New `style="modern"` concentric-ring layout as an alternative to the classic wheel. Five distinct rings — cusp/zodiac, ruler scale, planet clusters, house numbers, and aspect lines — for a clean, information-dense visualization. Works with all six themes and all chart types (Natal, Synastry, Transit, Lunar/Solar Return, Composite).
-   **Zodiac Ring Toggle**: New `show_zodiac_background_ring` parameter to remove colored zodiac wedges for a minimal look.
-   **`KerykeionChartStyle` Type**: New `Literal["classic", "modern"]` exported from `kerykeion.schemas`.

## New Features

### Modern Chart Style

Pass `style="modern"` to `save_svg()`, `generate_svg_string()`, `save_wheel_only_svg_file()`, or `generate_wheel_only_svg_string()` to use the new layout.

The modern layout renders 5 concentric rings:

1.  **Ring 1 (outer)**: House cusps with zodiac sign glyphs and degree/minute data
2.  **Ring 2**: Graduated ruler scale (1°/5°/10° ticks)
3.  **Ring 3**: Planet data clusters with indicator/tether lines
4.  **Ring 4 (inner)**: House numbers (1–12)
5.  **Ring 5 (core)**: Aspect lines with small glyphs at midpoints

```python
from kerykeion import AstrologicalSubjectFactory
from kerykeion.chart_data_factory import ChartDataFactory
from kerykeion.charts.chart_drawer import ChartDrawer

subject = AstrologicalSubjectFactory.from_birth_data(
    "John Lennon", 1940, 10, 9, 18, 30,
    lng=-2.9833, lat=53.4, tz_str="Europe/London", online=False,
)
chart_data = ChartDataFactory.create_natal_chart_data(subject)
chart = ChartDrawer(chart_data=chart_data)

# Modern full chart
chart.save_svg(style="modern")

# Modern wheel only
chart.save_wheel_only_svg_file(style="modern")

# Modern without zodiac ring
chart.save_svg(style="modern", show_zodiac_background_ring=False)
```

### Zodiac Background Ring Toggle

The `show_zodiac_background_ring` parameter (default `True`) controls whether colored zodiac wedges are drawn in the outer ring. Modern style only — ignored by the classic style.

### `KerykeionChartStyle` Type

New literal type exported from `kerykeion.schemas`:

```python
from kerykeion.schemas import KerykeionChartStyle
# KerykeionChartStyle = Literal["classic", "modern"]
```

### New Drawing Module

-   `kerykeion.charts.draw_modern` with `draw_modern_horoscope()` and `draw_modern_dual_horoscope()` functions.
-   New SVG template `kerykeion/charts/templates/modern_wheel.xml` for standalone modern wheel rendering.

## Bug Fixes

-   **Ruler ring ticks**: Now uniformly spaced across the full 360° circle.
-   **Planet cluster sizes**: Sub-element sizes increased and size progression reduced for better readability.

## Upgrade Path

This release is fully backward compatible with v5.10.x. The default `style="classic"` is unchanged — no migration needed. To use the new modern style, simply pass `style="modern"` to any chart rendering method.
