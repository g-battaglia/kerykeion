# 6.0.0a1

_2026-03-29_

## Highlights

- **22 new astrological features**, 8 new standalone factories, 11 new celestial points
- **Dual ephemeris backend**: libephemeris (pure Python, default) and swisseph (C bindings, optional)
- All features are opt-in -- existing code works unchanged

## New Standalone Factories

| Factory | Description |
|---------|-------------|
| PrimaryDirectionsFactory | Placidus semi-arc primary directions (Ptolemy + Naibod keys) |
| AstroCartographyFactory | Planetary line mapping (ACG) with configurable step/tolerance |
| EclipseFactory | Solar/lunar eclipse search (global + localized) |
| PlanetaryPhenomenaFactory | Phase angle, illumination, elongation, apparent magnitude |
| PlanetaryNodesFactory | Ascending/descending nodes, perihelion/aphelion |
| HeliacalFactory | Heliacal rising/setting with atmospheric parameters |
| OccultationFactory | Lunar occultation search (global + location-specific) |
| FixedStarDiscoveryFactory | Auto-discover fixed stars near natal planets |

## New Celestial Points

- 8 Uranian / Hamburg School hypothetical planets (Cupido, Hades, Zeus, Kronos, Apollon, Admetos, Vulkanus, Poseidon)
- 3 Lilith/Priapus variants (Interpolated Lilith, Mean Priapus, True Priapus)

## New Calculation Options

All via `AstrologicalSubjectFactory.from_birth_data()` kwargs:

- `calculate_dignities=True` -- Ptolemaic essential dignities
- `calculate_nakshatra=True` -- Vedic lunar mansions (27 Nakshatras)
- `calculate_gauquelin=True` -- Gauquelin 36-sector system with SVG rendering
- `calculate_local_space=True` -- Horizon coordinates (azimuth + altitude)
- `calculate_nutation=True` -- Earth nutation model
- `active_fixed_stars=[...]` -- Dynamic fixed star addition

## Ephemeris Backend Abstraction

Kerykeion now supports two interchangeable ephemeris backends:

| | libephemeris (default) | swisseph (optional) |
|---|---|---|
| Install | `pip install kerykeion` | `pip install kerykeion[swiss]` |
| Language | Pure Python | C bindings |
| License | AGPL-3.0 | GPL-2.0 |
| Source | NASA JPL DE440/DE441 | Swiss Ephemeris |
| Barycentric | Native JPL frame (more accurate) | Derived from heliocentric |

Both backends are 100% API-compatible. Planetary positions agree within 0.02 deg; house cusps within 0.05 deg. Select via `KERYKEION_BACKEND` env var or let auto-detection choose.

See `site/docs/backend_precision_comparison.md` for detailed precision analysis.
